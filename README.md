# 📊 Unemployment Analysis in India (EDA)

A comprehensive **Exploratory Data Analysis (EDA)** project on the **Unemployment in India** dataset using **Python**. This project focuses on data cleaning, preprocessing, visualization, and statistical analysis to uncover unemployment trends, regional patterns, and labour market insights.

---

## 📌 Project Overview

Unemployment is one of the key indicators of a country's economic health. This project explores unemployment data across different regions of India to identify trends, compare urban and rural unemployment, and analyze labour participation using various data visualization techniques.

The project demonstrates a complete EDA workflow, from data preprocessing to extracting meaningful business insights.

---

## 🎯 Objectives

- Explore the structure of the dataset.
- Identify and handle missing values.
- Analyze unemployment trends across regions.
- Compare Urban and Rural unemployment rates.
- Study labour participation and employment estimates.
- Visualize correlations among numerical features.
- Extract meaningful insights through statistical analysis.
